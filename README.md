common data analysis and presentation tasks for The Lab @ DC.

## Installation

You can install the development version of `{labtoolbox}` from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("thelabdc/labtoolbox")
```
